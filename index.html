Task Demonstrations

Task 1: Assembly from Scratch

Robot completes the entire assembly sequence without human intervention.

Task 2: Resume from Partial State

Robot resumes the assembly sequence from a predefined intermediate state, simulating a handover from a virtual human operator. The agent must perceive the current configuration and continue the task correctly.

Task 3: Error Detection & Recovery

Identifying incorrect parts and performing dynamic correction moves.
